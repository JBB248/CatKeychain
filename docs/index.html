--The Digital Cat Gallery--

This is a practice project I challenged myself with to improve my work efficiency, and my skills with HaxeFlixel, reactive UI, and API requests.

The cat gallery stores and displays photos and information about cats. Photos can be downloaded from TheCatAPI.

--What it does--

Here, photos can be viewed and downloaded from TheCatAPI. Before downloading, each photo can be assigned a nickname (used for the gallery's search feature) and a custom note.

If the photos provided aren't to your liking, they can be reshuffled using CTRL+R.

The gallery houses all of the photos in an organized and easy to navigate manner.

Use CTRL+F to activate the search function to find a specific photo using its given nickname.

--How to Use it--

Because this app uses an API with limited requests, I cannot distribute an executable version.

However, I can distribute the source code and building instructions on github.com
